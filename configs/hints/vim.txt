vim

NAVIGATION
  C-h/j/k/l move between splits/tmux panes
  gg/G      top/bottom of file
  :{n}      jump to line n
  {n}G      jump to line n
  H/M/L     top/middle/bottom of screen
  C-d/C-u   half-page down/up
  C-f/C-b   full page down/up
  %         jump to matching bracket
  {/}       prev/next paragraph
  [[/]]     prev/next section
WORD MOTIONS (󰆾)
  w/W       next word/WORD
  b/B       prev word/WORD
  e/E       end of word/WORD
  0/^/$     start/first char/end of line
  f{c}/F{c} find char fwd/back
  t{c}/T{c} till char fwd/back
  ;/,       repeat/reverse last f/t
SEARCH
  \//?       search fwd/back
  n/N       next/prev match
  */#       word under cursor fwd/back
EDITING
  ciw/caw   change inner/around word
  ci"/ci(   change inside quotes/parens
  di"/da(   delete inside/around parens
  yiw/yy    yank word/line
  p/P       paste after/before
  .         repeat last change
  u/C-r     undo/redo
VISUAL MODE
  v/V/C-v   char/line/block select
  gv        reselect last visual
SURROUND
  cs"'      change " to '
  ds"       delete surrounding "
  ysiw"     surround word with "
COMMENTARY
  gcc       comment/uncomment line
  gc{󰆾}     comment motion (gcap=paragraph)
  gc        comment selection (visual)
WINDOWS
  C-w·s     horizontal split
  C-w·v     vertical split
  C-w·o     close all other windows
  C-w·=     equalize sizes
MACROS & REGISTERS
  qa/q      record/stop macro into a
  @a/@@     play macro/replay last
  "ay/"ap   yank/paste register a
MARKS
  ma        set mark a
  'a/`a     jump to line/exact position
  ''        jump back
MISC
  >>/<<     indent/dedent line
  =         auto-indent (motion or visual)
  J         join lines
  ~         toggle case
  C-a/C-x   increment/decrement number
